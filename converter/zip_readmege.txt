Probleme nach zip Download 

Einige Browser (vor allem Microsoft Internet Explorer) "helfen" dem Benutzer in dem sie entpackte Dateien aus einer heruntergeladenen zip Datei blockieren.
Konsequenzen vom Blockieren:
*.chm Hilfedateien werden dann beim Anzeigen nicht vollständig angezeigt (enthalten aber in unserem Fall nur harmlose Hilfetexte )
*.exe Programmdateien werden vor dem Start mit einem Warnungsdialog gestartet.

Wenn Sie die Dateien von http://www.gnmidi.com oder http://www.gnmidi.at heruntergeladen haben, können sie diese auf nichts begründeten Warnungen ignorieren.

Mit dem Microsoft Explorer können Sie in den Eigenschaften zu diesen Dateien die Blockierungen aufheben, in dem Sie den Button Zulassen drücken im Allgemein Register.
