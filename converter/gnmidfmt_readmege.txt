GNMIDFMT 2 für Windows 7 und Windows 8

MIDI Format Konvertierung von einer oder vielen Dateien 
   liest Standard MIDI Dateien (Format 0 oder 1 oder 2)
   prüft Standard MIDI Dateien auf Fehler 
   konvertiert MIDI Format 0,1,2 in 0,1,2
   konvertiert RIFF MIDI .rmi in .mid Format 0, 1, 2
   konvertiert einzelne Datei
   konvertiert ausgewählte MIDI Dateien in einem Verzeichnis
   konvertiert alle MIDI Dateien in einem Verzeichnis

   einige Keyboards und Softwareprogramme können nur Format 0 lesen. Die meisten MIDI Dateien sind aber im MIDI Format 1 vorhanden. Mit diesem Programm ist das kein Problem mehr.
